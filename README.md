# Statistical Charts
Online Judge usage analysis visualization system.

## Features
+ React
+ React-router
+ Redux
+ Immutable
+ ES6
+ Yarn
+ Antd
+ Echarts
+ ......

## Usage
### Development
```
git clone https://github.com/ttop5/statistical-charts.git
cd statistical-charts
yarn install
yarn start
```
view `127.0.0.1:8000` on your browser
### Production
```
git clone https://github.com/ttop5/statistical-charts.git
cd statistical-charts
yarn install
yarn start:dist
```
